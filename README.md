# Mohamed Amr Portfolio

A modern personal portfolio built with React, Vite, and Tailwind CSS.

## Sections

- **About Me** — Personal info and contact details
- **Education** — Academic background
- **Skills** — Categorized skill cards
- **Experience** — Professional timeline
- **Project Experience** — Project cards with modal gallery

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Adding Project Images

Place your project images in `public/assets/projects/`:

| Project | Cover Image | Gallery Images |
|---------|-------------|----------------|
| GRC Product Development | `grc-cover.jpg` | `grc-1.jpg`, `grc-2.jpg`, `grc-3.jpg` |
| Badr University in Cairo | `buc-cover.jpg` | `buc-1.jpg`, `buc-2.jpg`, `buc-3.jpg` |
| GOEIC | `goeic-cover.jpg` | `goeic-1.jpg`, `goeic-2.jpg`, `goeic-3.jpg` |
| Chemistry Administration | `chemistry-cover.jpg` | `chemistry-1.jpg`, `chemistry-2.jpg`, `chemistry-3.jpg` |

Update project dates and tools in `src/data/portfolioData.js`.

## Build for Production

```bash
npm run build
```
